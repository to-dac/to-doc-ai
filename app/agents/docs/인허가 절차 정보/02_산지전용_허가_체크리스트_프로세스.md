# 산지전용 허가ㆍ변경허가 신청서

## 체크리스트 서류

### 산지전용허가 신청

- [ ] 사업계획서
  - 산지전용 목적
  - 사업기간
  - 산지전용을 하려는 산지의 이용계획
  - 입목ㆍ대나무 벌채를 통한 이용 또는 처리계획
  - 토석처리계획
  - 피해방지계획
- [ ] 산지전용타당성조사 결과서
  - 허가신청일 전 5년 이내 완료된 조사 결과서
- [ ] 산지의 소유권 또는 사용ㆍ수익권 증명서류
- [ ] 산지전용예정지가 표시된 지형도 또는 지적도
- [ ] 산지전용예정지 실측도
- [ ] 산림조사서
  - 수목이 있는 경우
  - 단, 일정 면적 미만 등 예외 가능
- [ ] 복구계획서
  - 복구대상 산지가 있는 경우
- [ ] 표고조사서 및 평균경사도조사서
- [ ] 농업인 증명 관련 서류
  - 해당하는 경우
- [ ] 재해위험성 검토의견서
  - 산지전용 면적 660㎡ 이상 5,000㎡ 미만인 경우 등
- [ ] 재해영향평가 협의 결과
  - 산지전용 면적 5,000㎡ 이상인 경우
- [ ] 재선충병방제계획서
  - 반출금지구역이 포함된 산지를 전용하는 경우

### 산지전용 변경허가 신청

- [ ] 변경사실을 증명할 수 있는 서류
- [ ] 변경 면적에 따라 필요한 산림조사서
- [ ] 표고조사서 및 평균경사도조사서
- [ ] 재해위험성 검토의견서
- [ ] 재해영향평가 협의 결과
  - 해당하는 경우

### 담당 공무원 확인사항

- [ ] 토지 등기사항증명서
- [ ] 축산업등록증
  - 농업인 증명이 필요한 경우

## 처리 프로세스
신청서 작성
→ 접수
→ 현지조사
→ 대체산림자원조성비 및 추가복구비 산정
→ 대체산림자원조성비 납부고지 및 추가복구비 예치 통지
→ 대체산림자원조성비 납부 및 추가복구비 예치
→ 허가증 작성
→ 허가증 발급
