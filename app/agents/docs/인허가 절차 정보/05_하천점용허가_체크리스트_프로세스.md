# 하천점용허가ㆍ홍수관리구역 안에서의 행위허가 신청서

## 체크리스트 서류

업로드된 신청서 양식 기준으로는 별도의 “첨부서류” 항목이 따로 분리되어 있지 않고, 신청서에 아래 내용을 구체적으로 작성하는 구조임.

- [ ] 신청인 정보
- [ ] 하천의 명칭
- [ ] 점용 또는 행위 위치
- [ ] 점용 또는 행위 면적
- [ ] 점용 또는 행위 목적
- [ ] 지구지정현황
- [ ] 점용 또는 행위 기간
- [ ] 변경내용
  - 변경신청인 경우만 작성

## 처리 프로세스
신청서 작성
→ 접수
→ 검토
→ 결재
→ 대장 정리
→ 허가증 작성
→ 허가증 발급
