# 도로점용허가 신청서

## 체크리스트 서류

- [ ] 설계도면
  - 전자도면으로 제출
- [ ] 주요지하매설물 관리자의 의견서
  - 도로 굴착을 수반하는 경우
- [ ] 주요지하매설물 사후관리계획
  - 신청인이 주요지하매설물 관리자인 경우
- [ ] 도로관리심의회 심의ㆍ조정 결과를 반영한 안전대책 서류
  - 도로 굴착을 수반하는 경우
- [ ] 수수료
  - 1,000원 또는 지방자치단체 조례로 정한 금액

## 처리 프로세스
신청서 작성
→ 도로관리청 접수
→ 점용 목적ㆍ장소ㆍ면적 검토
→ 굴착 수반 여부 확인
→ 필요 시 지하매설물 관리자 의견 확인
→ 필요 시 도로관리심의회 심의ㆍ조정
→ 점용 허가 여부 결정
