# 농지전용 허가ㆍ변경허가 신청서

## 체크리스트 서류

- [ ] 사업계획서
  - 전용목적
  - 사업시행자
  - 시행기간
  - 시설물 배치도
  - 소요자금 조달방안
  - 시설물 관리ㆍ운영계획
  - 대기환경ㆍ물환경 관련 사업장 규모
- [ ] 농지 소유권 입증서류
  - 토지 등기사항증명서로 확인할 수 없는 경우
- [ ] 사용승낙서 또는 사용권 입증서류
- [ ] 전용예정구역이 표시된 지적도등본
- [ ] 임야도등본
- [ ] 지형도
- [ ] 피해방지계획서
  - 농지개량시설 또는 도로 폐지ㆍ변경
  - 토사 유출
  - 폐수 배출
  - 악취 발생 등 인근 농지와 농어촌생활환경 피해가 예상되는 경우
- [ ] 변경사유서 및 변경내용 증명서류
  - 변경허가 신청의 경우
- [ ] 농지보전부담금 권리 양도양수 증명서류
  - 전용허가자 명의 변경 시
- [ ] 농지보전부담금 분할납부신청서
  - 분할납부 신청 시

### 담당 공무원 확인사항

- [ ] 해당 농지의 토지 등기사항증명서
- [ ] 지적도ㆍ임야도 및 지형도
