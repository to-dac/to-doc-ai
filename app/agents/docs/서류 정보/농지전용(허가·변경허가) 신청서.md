# 3. 농지전용(허가·변경허가) 신청서

- templateCode: `farmland_conversion_permit`
- name: `농지전용(허가·변경허가) 신청서`
- description: `농지법 시행규칙 별지 제14호서식 기반 폼 데이터`
- version: `2019-06-28`
- law: `농지법 시행규칙`
- formNo: `별지 제14호서식`
- processingDays:
    - 시군구: `10일`
    - 시도: `20일`
    - 농림축산식품부: `30일`

## 1. 기본 정보

- 접수번호 `receiptNumber`
- 접수일자 `receiptDate`
- 신청유형 `applicationType`
    - type: `CHOICE`
    - display: `radioGroup`
    - options:
        - 허가
        - 변경허가

## 2. 신청인

- 성명(명칭) `applicantName`
- 주민등록번호(법인등록번호) `applicantRegistrationNumber`
- 주소 `applicantAddress`
- 우편물수령지 `applicantMailAddress`
- 전화번호 `applicantPhone`

## 3. 전용 농지 소재지

- 소재지 `farmlandLocation`
- 번지 `farmlandLotNumber`
- 외 필지 수 `farmlandExtraLots`
    - unit: `필지`

## 4. 농지 면적 구분

### 농업진흥구역

- 농업진흥구역 계 `promotionZoneTotal`
    - unit: `㎡`
- 농업진흥구역 답 `promotionZonePaddyField`
    - unit: `㎡`
- 농업진흥구역 전 `promotionZoneDryField`
    - unit: `㎡`
- 농업진흥구역 농지개량시설부지 `promotionZoneFacilityLand`
    - unit: `㎡`

### 농업보호구역

- 농업보호구역 계 `protectionZoneTotal`
    - unit: `㎡`
- 농업보호구역 답 `protectionZonePaddyField`
    - unit: `㎡`
- 농업보호구역 전 `protectionZoneDryField`
    - unit: `㎡`
- 농업보호구역 농지개량시설부지 `protectionZoneFacilityLand`
    - unit: `㎡`

### 농업진흥지역 밖

- 농업진흥지역 밖 계 `outsideZoneTotal`
    - unit: `㎡`
- 농업진흥지역 밖 답 `outsideZonePaddyField`
    - unit: `㎡`
- 농업진흥지역 밖 전 `outsideZoneDryField`
    - unit: `㎡`
- 농업진흥지역 밖 농지개량시설부지 `outsideZoneFacilityLand`
    - unit: `㎡`

### 합계

- 합계 계 `grandTotal`
    - unit: `㎡`
- 합계 답 `grandTotalPaddyField`
    - unit: `㎡`
- 합계 전 `grandTotalDryField`
    - unit: `㎡`
- 합계 농지개량시설부지 `grandTotalFacilityLand`
    - unit: `㎡`

## 5. 사업 정보 및 전용목적

- 사업예정부지 총 면적(비농지 포함) `totalProjectArea`
    - unit: `㎡`
- 농업진흥지역 면적 `promotionArea`
    - unit: `㎡`
- 착공예정일 `constructionStartDate`
- 준공예정일 `constructionEndDate`
- 전용목적 `conversionPurpose`

## 6. 농지보전부담금 납부방법

- 농지보전부담금 납부방법 `levyPaymentMethod`
    - type: `CHOICE`
    - display: `radioGroup`
    - options:
        - 일시납부
        - 분할납부

## 7. 전용신청농지명세서

- 전용신청농지명세서 `farmlandDetailsTable`
    - type: `TABLE`
    - description: 신청 대상 농지 필지별 상세 정보를 입력합니다.
    - columns:
        - 시·군 `city`
        - 읍·면 `town`
        - 리·동 `village`
        - 지번 `lotNumber`
        - 지목 `landCategory`
        - 면적 `area`
            - unit: `㎡`
        - 진흥지역 용도구분 `promotionZoneType`
        - 전용면적 `conversionArea`
            - unit: `㎡`
        - 주재배 작물명 `mainCrop`

## 8. 신청인 서명

- 신청일 `applicationDate`
- 신청인 서명 또는 인 `applicantSignature`
