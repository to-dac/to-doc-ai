# 5. 하천 점용 허가 신청서

- templateCode: `river_permit`
- name: `하천 점용 허가 신청서`
- description: `하천법 시행규칙 [별지 제29호서식] <개정 2025. 10. 1.>`
- version: `2025-10-11`

## 1. 기본 정보

- 접수번호 `receiptNumber`
- 접수일 `receiptDate`
- 허가 종류 `permitType`
    - type: `CHOICE`
    - display: `radioGroup`
    - options:
        - 하천점용허가
        - 홍수관리구역 안에서의 행위허가
- 신청 구분 `applicationCategory`
    - type: `CHOICE`
    - display: `radioGroup`
    - options:
        - 신규
        - 변경

## 2. 신청인

- 성명(법인인 경우 그 명칭 및 대표자의 성명) `applicantName`
- 생년월일(법인등록번호) `applicantBirthRegistration`
- 주소 `applicantAddress`
- 전화번호 `applicantPhone`

## 3. 점용(행위) 개요

- 하천의 명칭 `riverName`
- 점용(행위) 위치 `occupancyLocation`
- 면적 `occupancyArea`
    - unit: `㎡`
- 점용(행위) 목적 `occupancyPurpose`
    - 설명: 하천법 제33조제1항 각 호 및 시행령 제35조제1항 각 호의 행위별 명세를 구체적으로 작성
- 지구지정현황(하천법 제44조) `districtDesignation`
- 점용(행위)기간 시작일 `occupancyStartDate`
- 점용(행위)기간 종료일 `occupancyEndDate`
- 점용(행위) 기간(일) `occupancyDays`
    - unit: `일`

## 4. 변경내용

- 변경내용 `changeContent`
    - 설명: 허가사항에 대하여 변경신청 할 경우에만 작성

## 5. 신청인 서명

- 신청일 `applicationDate`
- 신청인 서명 또는 인 `applicantSignature`
