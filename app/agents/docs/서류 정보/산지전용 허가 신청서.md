## 1. 기본 정보

- 접수번호 `receiptNumber`
- 접수일 `receiptDate`
    - type: `DATE`
    - display: `date`
- 신청유형 `applicationType`
    - type: `CHOICE`
    - display: `radioGroup`
    - options:
        - 허가
        - 변경허가

## 2. 신청인

- 성명 `applicantName`
- 주민등록번호 `applicantResidentNumber`
- 주소 `applicantAddress`
    - display: `textarea`
- 전화번호 `applicantPhone`
- 해당 산지에 대한 권리관계 `applicantRights`
    - display: `textarea`

## 3. 산지 소유자

- 소유자 성명 `ownerName`
- 소유자 생년월일 `ownerBirthDate`
    - type: `DATE`
    - display: `date`
- 소유자 주소 `ownerAddress`
    - display: `textarea`
- 소유자 전화번호 `ownerPhone`

## 4. 전용대상 산지

- 전용대상 산지 명세 `conversionTargetTable`
    - type: `TABLE`
    - display: `table`
    - 설명: 전용 신청 산지 필지별 소재지·면적 정보를 입력합니다
    - columns:
        - 소재지 `location`
        - 지번 `lotNumber`
        - 지목 `landCategory`
        - 면적 계 `totalArea`
            - unit: `㎡`
        - 임업용산지 `forestryArea`
            - unit: `㎡`
        - 공익용산지 `publicArea`
            - unit: `㎡`
        - 준보전산지 `conservationArea`
            - unit: `㎡`
